
# Dashboard de Vendas — Xbox Game Pass

Este projeto apresenta um dashboard para análise de vendas de assinaturas do Xbox Game Pass, transformando dados brutos em informações visuais para apoiar decisões.

## Objetivo
Organizar e visualizar dados de assinaturas, destacando faturamento por tipo de plano, auto-renovação e vendas de passes adicionais.

## Estrutura
- Dados incluem: ID, Nome, Plano (Core, Standard, Ultimate), Tipo de Assinatura (Mensal, Trimestral, Anual), Auto-renovação, Preços e Total consolidado.
- Informações sobre EA Play Season Pass e Minecraft Season Pass.

## Indicadores
- Faturamento total de planos anuais: 1.754
- Faturamento anual por auto-renovação: Sim = 1.537 | Não = 217
- Vendas EA Play Season Pass: 600
- Vendas Minecraft Season Pass: 940

## Dashboard
- Título: XBOX GAME PASS SUBSCRIPTIONS SALES
- Período: 01/01/2025 a 10/12/2025
- Segmentações para filtros por plano, tipo e auto-renovação.

## Paleta de Cores
Baseada na identidade Xbox: tons de verde (#9BC848, #22C55E) e complementares.

## Como usar
1. Abra o arquivo no Excel (2010+).
2. Acesse a aba Dashboard para visualizar KPIs e gráficos.
3. Utilize segmentações para filtrar dados.
